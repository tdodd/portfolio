    MeetPanda

    This small Node.js web application uses your current location to find meetups. Specify a radius and see what’s going on around you.

    HTML5’s geolocation functionality allows the app to gain access to the client’s location. The Google Maps API then provides a widget for choosing a radius. With this data, the meetup API can retrieve a list of meetups, which is then presented to the client.

  • 2017
    DriveShark

    Get notifications about travel times when you want them.

    2017
    x
    DriveShark

    Built with Python and the Django framework, DriveShark lets you add your common trips, and specify when to receive notifications about travel times to these locations. The user adds a starting point and a destination. This is then saved as a trip, which the user can request to be notified about when they desire.

    Postgresql is the DBMS, while Django provides an MVC structure, as well as a templating engine for the views.

  • May - Sept 2016
    Iberian Diaspora Research Project

    Department of Modern Languages, University of Ottawa

    Full-stack development of a bilingual web application using the Laravel PHP framework.

    May - Sept 2016
    x
    Iberian Diaspora Research Project

    This research project is an archive of Spanish and Portuguese speaking migrants from the Iberian peninsula between the 15th and 17th centuries. It contains information about their relationships to other migrants, occupations, and the sources used to obtain this information, among other personal details.

    The application has a client-facing portal, allowing users to perform searches based on location, occupation, date of birth, etc. It also has an admin side, that allows researchers to log in and enter data, source information, relationships, and filter the results. This was all implemented using the excellent PHP framework that is Laravel!
